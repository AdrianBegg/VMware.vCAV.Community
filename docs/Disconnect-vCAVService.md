# Disconnect-vCAVService

## SYNOPSIS
This cmdlet logs off the currently connected vCloud Availability service.

## SYNTAX

```
Disconnect-vCAVService
```

## DESCRIPTION
This cmdlet logs off the currently connected vCloud Availability service.

## EXAMPLES

### EXAMPLE 1
```
Disconnect-vCAVService
```

Disconnects the currently configured session in the $global:DefaultvCAVServer global variable.
